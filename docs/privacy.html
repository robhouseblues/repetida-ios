Política de Privacidade

Repetida · Atualizado em 9 de junho de 2026

Repetida (“o app”) é um organizador de álbum de figurinhas. Esta política descreve como tratamos informações quando você usa o app.

Resumo

  • Não pedimos conta, login ou cadastro.
  • Seus dados de coleção ficam apenas no seu iPhone.
  • Não vendemos nem compartilhamos seus dados com terceiros.
  • O app funciona offline e não exige conexão com a internet.

Dados que o app armazena

O app salva localmente no dispositivo informações sobre sua coleção, por exemplo: figurinhas marcadas como possuídas, quantidade de repetidas e histórico recente de alterações. Esses dados servem somente para o funcionamento do app.

Dados que não coletamos

  • Nome, e-mail ou identificadores de conta
  • Localização
  • Contatos ou fotos (exceto quando você escolhe exportar/compartilhar conteúdo pelo iOS)
  • Dados de uso enviados a servidores nossos

Compartilhamento

Quando você usa “Compartilhar” (por exemplo, lista de repetidas ou imagem para WhatsApp), o conteúdo é enviado pelos recursos nativos do iOS para o destino que você escolher. Nós não recebemos nem armazenamos essas exportações.

Crianças

O app não coleta dados pessoais de forma intencional. Pais e responsáveis podem usar o app para acompanhar o álbum de crianças no próprio dispositivo.

Alterações

Podemos atualizar esta política ocasionalmente. A data no topo indica a versão vigente.
